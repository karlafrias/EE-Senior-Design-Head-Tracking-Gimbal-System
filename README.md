## Repository Contents

**3D Models:** Contains 3D CAD models for the head-tracking system and related mechanical components created in Fusion 360. Also includes files used for 3D printing and system integration.

**Code:** Contains the embedded code used for the head-tracking FPV camera control system, including communication and control logic for the headset and camera system.

**Datasheets:** Contains manufacturer datasheets for key project components used in the system design and integration.

**PCB Files:** Contains the PCB design files created in Altium, along with fabrication outputs, gerber files, and other board-related design documents.

**Pin Diagrams:** Contains wiring references, pin mappings, and connection diagrams for the major components used in the project.

**Poster-Slides:** Contains our senior design poster, presentation slides, and supporting materials used to present the project.

**Technical Report:** Contains the written technical report and supporting documentation for the project.

## Project Abstract

Inspired by Max Imagination and redesigning his board, our senior design project focuses on developing a custom head-tracking FPV camera control system for real-time remote viewing applications. The goal of this project is to create a more refined and functional board that supports directional camera control through natural head movement. By redesigning the board, the team aimed to improve organization, functionality, and overall integration while supporting future use in applications such as drones, remote sensing, and first responder assistance.

Our project involved designing the PCB in Altium, developing 3D CAD models in Fusion 360, selecting and sourcing components, assembling and soldering the board, testing power and signal behavior, and integrating the board with the camera and headset system. Along the way, the team addressed engineering challenges such as axis inversion, signal routing, communication reliability, mechanical fit, and hardware integration.
